Fundamentos de Javascript

Introducción​

JavaScript es un lenguaje de programación de alto nivel, interpretado y orientado a objetos que se utiliza para crear aplicaciones web interactivas y dinámicas. Fue creado en 1995 por Brendan Eich mientras trabajaba en Netscape Communications Corporation.

Los fundamentos de JavaScript son los conceptos básicos que debes entender para poder utilizar el lenguaje de manera efectiva. Estos incluyen:

Variables y tipos de datos: las variables se utilizan para almacenar valores en JavaScript, y existen varios tipos de datos que se pueden almacenar en ellas, como números, cadenas de texto, booleanos, objetos y arreglos.

Operadores: los operadores se utilizan para realizar operaciones en JavaScript, como sumar números, concatenar cadenas de texto, comparar valores y asignar valores a variables.

Estructuras de control de flujo: las estructuras de control de flujo se utilizan para controlar el flujo de ejecución de un programa JavaScript. Estas incluyen estructuras como if-else, while, for, switch y try-catch.

Funciones: las funciones se utilizan para encapsular un bloque de código que realiza una tarea específica, lo que permite reutilizar ese bloque de código en diferentes partes de un programa. Las funciones pueden aceptar argumentos y devolver valores.

Objetos y clases: los objetos se utilizan para representar entidades del mundo real en un programa JavaScript, como usuarios, productos, pedidos, etc. Las clases son plantillas para crear objetos que comparten las mismas propiedades y métodos.

Manipulación del DOM: el Document Object Model (DOM) es una representación en forma de árbol de un documento HTML que se puede manipular con JavaScript para cambiar el contenido de una página web en tiempo real.

Mi primer "Hola mundo" en Javascript​

console.log("Hola, mundo!");

Este código imprimirá "Hola, mundo!" en la consola del navegador o en la consola de desarrollo de tu entorno de ejecución de JavaScript.

Formas de implementar Javascript​

  1. Etiqueta <script> en el archivo HTML:
    Puedes incluir tu código JavaScript directamente dentro de las etiquetas <script> en el archivo HTML. Puedes colocar el código en la sección <head> o antes de cerrar el <body> para asegurarte de que el código se ejecute después de que la página se haya cargado.
<!DOCTYPE html>
<html>
<head>
  <title>Mi Sitio Web</title>
  <script>
    // Tu código JavaScript aquí
    alert("Hola, mundo!");
  </script>
</head>
<body>
  <!-- Contenido de tu sitio web -->
</body>
</html>

  1. Archivo externo JavaScript:
    También puedes colocar tu código JavaScript en un archivo externo con extensión .js y luego enlazarlo en tu archivo HTML utilizando la etiqueta <script> con el atributo src que especifica la ubicación del archivo externo.

Archivo script.js:

// Contenido del archivo script.js
alert("Hola, mundo!");

Archivo HTML:

<!DOCTYPE html>
<html>
<head>
  <title>Mi Sitio Web</title>
  <script src="script.js"></script>
</head>
<body>
  <!-- Contenido de tu sitio web -->
</body>
</html>

Variables:​

Las variables son como contenedores que se utilizan para almacenar valores en JavaScript. Se pueden declarar variables utilizando la palabra clave var, let o const, seguido de un nombre de variable y, opcionalmente, un valor inicial.

Por ejemplo, para declarar una variable llamada mensaje y asignarle el valor "Hola, mundo!", puedes usar la siguiente sintaxis:

var mensaje = "Hola, mundo!";

En este ejemplo, la palabra clave var se utiliza para declarar la variable mensaje y se le asigna el valor de la cadena "Hola, mundo!".

Diferencias entre var, let y const​

var:
var es la forma más antigua de declarar variables en JavaScript. La declaración de una variable con var tiene un alcance de función o alcance global, dependiendo de si se declara dentro o fuera de una función. Las variables declaradas con var se elevan al inicio de su ámbito, lo que significa que se pueden acceder antes de su declaración. Una variable declarada con var puede ser reasignada y su valor puede ser modificado. No tiene restricciones en cuanto a la redeclaración de una variable dentro del mismo ámbito.

let:
let fue introducido en ECMAScript 6 (ES6) y es la forma recomendada de declarar variables en JavaScript a partir de esa versión. La declaración de una variable con let tiene un alcance de bloque, lo que significa que solo está disponible dentro del bloque donde se declaró. Las variables declaradas con let no se elevan al inicio de su ámbito, por lo que no se pueden acceder a ellas antes de su declaración. Una variable declarada con let puede ser reasignada, lo que significa que su valor puede ser modificado. No se permite la redeclaración de una variable dentro del mismo ámbito utilizando let.

const:
const también fue introducido en ES6 y se utiliza para declarar variables cuyos valores no cambiarán una vez asignados. La declaración de una variable con const tiene un alcance de bloque al igual que let. Las variables declaradas con const deben inicializarse con un valor en el momento de la declaración y no se pueden asignar a un nuevo valor posteriormente. Aunque el valor de una variable const no puede cambiar, si es un objeto o un arreglo, las propiedades o elementos internos pueden ser modificados. Al igual que con let, no se permite la redeclaración de una variable utilizando const dentro del mismo ámbito.

Cuándo utilizar cada uno:​

var se puede usar si se necesita una variable con alcance global o si se está trabajando en un proyecto antiguo que aún no ha adoptado las características de ES6.
let se recomienda para la mayoría de los casos, ya que proporciona un alcance más estricto y evita problemas de reasignación accidental.
const debe usarse cuando se desea asignar un valor que no cambiará a lo largo del programa. Es especialmente útil para valores constantes o referencias a objetos inmutables.
En resumen, "var" tiene un alcance de función o global, "let" tiene un alcance de bloque y permite la reasignación, y "const" también tiene un alcance de bloque pero no permite la reasignación.

Tipos de datos:​

En JavaScript, hay varios tipos de datos que se pueden utilizar para almacenar valores en variables. Estos incluyen:

  • Números: se utilizan para almacenar valores numéricos, como 1, 2.5, -3, etc.
let numeroEntero = 10;
let numeroDecimal = 3.14;
let numeroNegativo = -5;
  • Cadenas de texto: se utilizan para almacenar valores de texto, como "Hola" o "Mundo".
let saludo = "Hola";
let nombre = "Mundo";
let frase = "Mi nombre es Carlos";
  • Booleanos: se utilizan para almacenar valores verdaderos o falsos.
let verdadero = true;
let falso = false;
  • Null: se utiliza para indicar que una variable no tiene un valor asignado..
let variableNula = null;
  • Undefined: se utiliza para indicar que una variable no está definida.
let variableIndefinida;
  • Objetos: se utilizan para almacenar colecciones de valores relacionados y funciones.
let persona = {
  nombre: "Juan",
  edad: 25,
  pais: "Argentina"

};

let libro = {
  titulo: "El Gran Gatsby",
  autor: "F. Scott Fitzgerald",
  año: 1925
};
  • Arreglos: se utilizan para almacenar colecciones de valores relacionados en una lista ordenada.
let numeros = [1, 2, 3, 4, 5];
let colores = ["rojo", "verde", "azul"];
let mezclado = [1, "dos", true, null];
