Introducción a Bootstrap

¿Qué es Bootstrap?​

Bootstrap es un framework de código abierto desarrollado por Twitter que facilita la creación de sitios web y aplicaciones web. Este framework está basado en HTML, CSS y JavaScript, y proporciona un conjunto de herramientas, estilos predefinidos y componentes reutilizables que permiten crear interfaces modernas y responsivas de manera rápida y sencilla.

Instalación​

Para comenzar a utilizar Bootstrap en tu proyecto, hay varias formas de incluirlo:

Descarga e incorporación local​

  1. Visita el sitio oficial de Bootstrap en getbootstrap.com.

  2. Haz clic en el botón de descarga para obtener el archivo ZIP de Bootstrap.

  3. Descomprime el archivo ZIP y copia las carpetas "css" y "js" en la ubicación de tu proyecto.

  4. Enlaza los archivos CSS y JS de Bootstrap en el documento HTML:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Mi Proyecto con Bootstrap</title>
  <!-- Enlace a los archivos CSS de Bootstrap -->
  <link rel="stylesheet" href="ruta_a_tu_archivo_css/bootstrap.min.css">
</head>
<body>
  <!-- Contenido del sitio -->
  
  <!-- Enlace a los archivos JS de Bootstrap (jQuery primero, luego Popper.js y finalmente Bootstrap) -->
  <script src="ruta_a_tu_archivo_js/jquery.min.js"></script>
  <script src="ruta_a_tu_archivo_js/popper.min.js"></script>
  <script src="ruta_a_tu_archivo_js/bootstrap.min.js"></script>
</body>
</html>

Utilizar un CDN (Content Delivery Network)​

Bootstrap también se puede usar a través de un CDN, lo que te permite enlazar los archivos directamente desde la web sin necesidad de descargarlos ni alojarlos en tu servidor. Esto puede facilitar el desarrollo y la implementación, especialmente si estás probando o prototipando un sitio rápidamente.

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Mi Proyecto con Bootstrap</title>
  <!-- Enlace a los archivos CSS de Bootstrap a través del CDN -->
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css">
</head>
<body>
  <!-- Contenido del sitio -->
  
  <!-- Enlace a los archivos JS de Bootstrap (jQuery primero, luego Popper.js y finalmente Bootstrap) a través del CDN -->
  <script src="https://code.jquery.com/jquery-3.5.1.slim.min.js"></script>
  <script src="https://cdn.jsdelivr.net/npm/@popperjs/core@2.9.1/dist/umd/popper.min.js"></script>
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.min.js"></script>
</body>
</html>

Grid System​

Una de las características más importantes de Bootstrap es su sistema de cuadrícula (grid system). Este sistema permite organizar el contenido en filas y columnas, lo que facilita la creación de diseños flexibles y responsivos.

Filas (Rows) y Columnas (Columns)​

El sistema de cuadrícula se basa en un sistema de 12 columnas. Cada fila (row) debe estar contenida dentro de un elemento con la clase "container" o "container-fluid". Dentro de cada fila, puedes agregar columnas utilizando las clases "col-" seguidas del número de columnas que deseas ocupar.

Por ejemplo, para dividir una fila en dos columnas iguales:

<div class="container">
  <div class="row">
    <div class="col-6">Columna 1</div>
    <div class="col-6">Columna 2</div>
  </div>
</div>

En este ejemplo, las dos columnas ocuparán cada una la mitad del ancho de la fila, ya que cada una tiene la clase "col-6" (6 columnas de las 12 disponibles).

Tamaños de Columnas Responsivas​

Bootstrap también ofrece clases para definir tamaños de columnas responsivas, lo que permite adaptar el diseño según el tamaño de la pantalla. Por ejemplo:

  • col-sm: Tamaño de columna para pantallas pequeñas (mínimo 576px).
  • col-md: Tamaño de columna para pantallas medianas (mínimo 768px).
  • col-lg: Tamaño de columna para pantallas grandes (mínimo 992px).
  • col-xl: Tamaño de columna para pantallas extra grandes (mínimo 1200px).

Ejemplo:

<div class="container">
  <div class="row">
    <div class="col-sm-6 col-md-4 col-lg-3 col-xl-2">Contenido</div>
    <div class="col-sm-6 col-md-4 col-lg-3 col-xl-2">Contenido</div>
    <div class="col-sm-6 col-md-4 col-lg-3 col-xl-2">Contenido</div>
    <div class="col-sm-6 col-md-4 col-lg-3 col-xl-2">Contenido</div>
    <div class="col-sm-6 col-md-4 col-lg-3 col-xl-2">Contenido</div>
    <div class="col-sm-6 col-md-4 col-lg-3 col-xl-2">Contenido</div>
  </div>
</div>

En este ejemplo, en pantallas pequeñas (col-sm), cada columna ocupará la mitad del ancho de la fila. En pantallas medianas (col-md), cada columna ocupará un cuarto del ancho de la fila. En pantallas grandes (col-lg) y extra grandes (col-xl), cada columna ocupará un sexto del ancho de la fila.

Componentes​

Bootstrap ofrece una variedad de componentes predefinidos que te ayudarán a crear interfaces web de manera rápida y profesional. Algunos de los componentes más comunes son:

Barra de navegación (Navbar)​

La barra de navegación es un componente que te permite crear menús de navegación responsivos.

<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <a class="navbar-brand" href="#">Mi Sitio</a>
  <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation">
    <span class="navbar-toggler-icon"></span>
  </button>
  <div class="collapse navbar-collapse" id="navbarNav">
    <ul class="navbar-nav">


      <li class="nav-item active">
        <a class="nav-link" href="#">Inicio</a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#">Acerca</a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#">Contacto</a>
      </li>
    </ul>
  </div>
</nav>

Botones (Buttons)​

Los botones de Bootstrap vienen en diferentes estilos y tamaños.

<button type="button" class="btn btn-primary">Botón Principal</button>
<button type="button" class="btn btn-secondary">Botón Secundario</button>
<button type="button" class="btn btn-success">Éxito</button>
<button type="button" class="btn btn-danger">Error</button>
<button type="button" class="btn btn-warning">Advertencia</button>
<button type="button" class="btn btn-info">Información</button>
<button type="button" class="btn btn-light">Light</button>
<button type="button" class="btn btn-dark">Dark</button>

Formularios (Forms)​

Bootstrap ofrece estilos predefinidos para crear formularios.

<form>
  <div class="form-group">
    <label for="nombre">Nombre:</label>
    <input type="text" class="form-control" id="nombre" placeholder="Ingresa tu nombre">
  </div>
  <div class="form-group">
    <label for="correo">Correo Electrónico:</label>
    <input type="email" class="form-control" id="correo" placeholder="Ingresa tu correo electrónico">
  </div>
  <button type="submit" class="btn btn-primary">Enviar</button>
</form>

Tarjetas (Cards)​

Las tarjetas son componentes versátiles que pueden contener contenido diverso.

<div class="card" style="width: 18rem;">
  <img src="imagen.jpg" class="card-img-top" alt="Imagen">
  <div class="card-body">
    <h5 class="card-title">Título de la Tarjeta</h5>
    <p class="card-text">Contenido de la tarjeta.</p>
    <a href="#" class="btn btn-primary">Más información</a>
  </div>
</div>

Conclusiones​

Bootstrap es una herramienta poderosa que facilita el desarrollo de sitios web y aplicaciones web. Su sistema de cuadrícula, componentes y estilos predefinidos hacen que el proceso de diseño y desarrollo sea más eficiente y rápido.
