¡Bienvenidos!

Lo que aprenderás​

¡Aprende a crear sitios web desde cero en Instituto Salvador! En este curso aprenderás las habilidades necesarias para desarrollar sitios web funcionales utilizando HTML, CSS, Bootstrap y Javascript. Además, aprenderás a utilizar Git y GitHub para controlar versiones de tus proyectos y colaborar con otros desarrolladores. Aprenderás a crear repositorios, hacer commits, crear ramas y fusionar cambios. Este curso está dirigido tanto a principiantes como a aquellos con experiencia previa en desarrollo web. ¡Comencémos en el emocionante mundo del desarrollo web en Instituto Salvador!

Plan de estudio del curso​

Módulo 1: Fundamentos de la web y HTML

  • ¿Qué es la web y cómo funciona?
  • Introducción a HTML: etiquetas, atributos y estructura de una página web
  • Creación de una página web simple utilizando HTML

Módulo 2: CSS

  • Introducción a CSS: selectores, reglas y propiedades
  • Maquetación y diseño de páginas web utilizando CSS
  • Introducción a Bootstrap: componentes y utilización

Módulo 3: JavaScript fundamentos

  • Variables y tipos de datos
  • Operadores y expresiones
  • Condicionales
  • Bucles

Módulo 4: JavaScript avanzado

  • Arrays y objetos
  • Funciones
  • Eventos y manipulación del DOM
  • Utilización de JavaScript y Bootstrap para crear una página web interactiva

Módulo 5: Git y Github

  • Introducción a Git y Github
  • Configuración de Git y Github
  • Creación y colaboración en repositorios en Github

Módulo 6: Proyecto final

  • Los estudiantes trabajarán en equipos para crear un proyecto final utilizando HTML, CSS, JavaScript, Bootstrap y Git/Github
  • Presentación y evaluación de proyectos finales

Requisitos​

  • Conocimientos básicos de computación
  • Conocimientos básicos de inglés (no excluyente)

¿Qué necesito para tomar este curso?​

  • Una computadora con acceso a internet
  • Un navegador web (puedes usar el de tu preferencia, te recomendamos Firefox Developer para seguir este curso)
  • Un editor de código (En este curso usaremos Visual Studio Code)
  • También usaremos las siguientes extenciones de Visual Studio Code:
    • Live Server
    • Prettier - Code formatter
    • Auto Close Tag
    • Auto Rename Tag
    • Path Intellisense

¿Qué obtendré al finalizar el curso?​

  • Habrás aprendido a crear sitios web funcionales utilizando HTML, CSS, Bootstrap y Javascript
  • Habrás aprendido a utilizar Git y GitHub para controlar versiones de tus proyectos y colaborar con otros desarrolladores
  • Habrás creado un proyecto final utilizando HTML, CSS, JavaScript, Bootstrap y Git/Github
