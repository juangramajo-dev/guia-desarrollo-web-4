¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje utilizado para dar estilo y diseño a páginas web. Es una herramienta crucial para cualquier diseñador o desarrollador web que quiera crear sitios web atractivos y funcionales.

Con CSS, los diseñadores pueden controlar la apariencia de elementos individuales en una página web, como el tamaño del texto, el color de fondo, la posición y el espaciado. También pueden aplicar estilos a grupos de elementos para crear diseños cohesivos y consistentes.

Sintaxis CSS​

La sintaxis de CSS consiste en reglas que especifican cómo se deben aplicar los estilos a los elementos en una página web. Cada regla consta de un selector y un bloque de declaraciones. Los selectores identifican los elementos a los que se aplicará el estilo, mientras que las declaraciones especifican qué estilo se aplicará.

Selectores de CSS​

Los selectores de CSS pueden ser muy específicos, como seleccionar un elemento por su clase o ID, o más generales, como seleccionar todos los elementos dentro de un contenedor. Como desarrolladores podemos utilizar múltiples selectores para aplicar estilos a grupos específicos de elementos.

Selectores básicos​

SelectorDescripción
*Selecciona todos los elementos del DOM
etiquetaSelecciona todas las etiquetas indicadas
.classSelección de los elementos con la clase .class
#idSelección del elemento con id #id
sel1 sel2Selección de los selectores sel2 que se encuentren dentro de los selectores sel1
.class1.class2Selección de los elementos con las dos clases: class1 y class2
sel1.class1Selección de todos los selectores sel1 con clase class1
sel1, sel2Selección de todos los selectores separados por comas
sel1 > sel2Selección de los selectores sel2 cuando son hijos de sel1
sel1 + sel2Selección del selector sel2 cuando es hermano de sel1 (su elemento padre es el mismo)

Propiedades y Valores de CSS​

Las propiedades de CSS son los atributos que se aplican a los elementos en una página web, como el color de fondo, la fuente y el tamaño del texto. Cada propiedad tiene un valor asociado que determina cómo se aplicará esa propiedad al elemento.

Hay muchas propiedades diferentes en CSS, desde las básicas como color y tamaño de fuente hasta las más avanzadas como animación y transformación.

Algunas propiedades CSS comunes incluyen:​

  • Color: color
  • Fondo: background-color
  • Fuente: font-family
  • Tamaño de fuente: font-size
  • Alineación: text-align
  • Márgenes: margin
  • Padding: padding
  • Bordes: border
  • Ancho: width
  • Altura: height

Un ejemplo de regla CSS es:​

p {
  color: blue;
  font-family: Arial;
  font-size: 16px;
}

Este ejemplo aplica a todos los elementos <p> en una página web un color azul, una fuente Arial y un tamaño de fuente de 16 píxeles.

Formas de agregar CSS​

Hay tres formas principales de incluir CSS en tu página web: mediante un archivo externo, en línea dentro de las etiquetas HTML y dentro de la sección de estilo en la misma página HTML.

  1. Archivo externo de CSS: Esta es la forma más común de incluir CSS en una página web. En este método, se crea un archivo CSS separado con todo el código CSS y se lo vincula a la página HTML utilizando la etiqueta <link> en la sección <head> del HTML.
<!DOCTYPE html>
<html>
  <head>
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>
    <!-- Contenido de la página aquí -->
  </body>
</html>

En este ejemplo, el archivo styles.css contiene todo el código CSS. La etiqueta <link> vincula el archivo CSS a la página HTML.

  1. CSS en línea: Esta forma de incluir CSS se realiza directamente en la etiqueta HTML utilizando el atributo style. En este caso, el código CSS se encuentra directamente dentro de las etiquetas HTML y se aplica solo a ese elemento específico.
<!DOCTYPE html>
<html>
  <head>
  </head>
  <body>
    <h1 style="color: red; font-size: 36px;">Título</h1>
    <p style="color: blue; font-size: 24px;">Párrafo</p>
  </body>
</html>

En este ejemplo, el código CSS se encuentra dentro del atributo style de las etiquetas HTML <h1> y <p>. El código CSS dentro del atributo style se aplica solo a esas etiquetas HTML específicas.

  1. CSS dentro de la sección de estilo: Esta forma de incluir CSS se realiza dentro de la sección de estilo en la misma página HTML. En este caso, el código CSS se encuentra en la sección <head> del HTML, dentro de las etiquetas <style>.
<!DOCTYPE html>
<html>
  <head>
    <style>
      h1 {
        color: red;
        font-size: 36px;
      }
      p {
        color: blue;
        font-size: 24px;
      }
    </style>
  </head>
  <body>
    <h1>Título</h1>
    <p>Párrafo</p>
  </body>
</html>

En este ejemplo, el código CSS se encuentra dentro de la etiqueta <style> en la sección <head>del HTML. El código CSS se aplica a las etiquetas HTML <h1> y <p>.

Nota: Es importante tener en cuenta que cada forma de incluir CSS tiene sus ventajas y desventajas. Por ejemplo, incluir CSS en un archivo externo facilita la modificación del código CSS y su mantenimiento. Sin embargo, a veces puede ser más conveniente utilizar CSS en línea si solo necesitas aplicar un estilo a un elemento específico. Y, por último, el uso de CSS dentro de la sección de estilo puede ser útil si solo necesitas aplicar un estilo a una página específica.
