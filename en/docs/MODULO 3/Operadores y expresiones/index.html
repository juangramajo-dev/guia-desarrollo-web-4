Operadores y expresiones

Introducción​

Los operadores y expresiones son elementos fundamentales en JavaScript que se utilizan para realizar operaciones y cálculos en el código. Los operadores son símbolos o palabras reservadas que se utilizan para indicar una acción a realizar, como sumar, restar, comparar valores, etc. Las expresiones son combinaciones de valores, variables, operadores y llamadas a funciones que se utilizan para producir un resultado.

En JavaScript existen diferentes tipos de operadores, como los aritméticos (para realizar operaciones matemáticas), los de asignación (para asignar valores a variables), los de comparación (para comparar valores), los lógicos (para evaluar expresiones booleanas) y los ternarios (para realizar una evaluación condicional).

Por su parte, las expresiones pueden ser simples o complejas, y pueden combinar varios operadores y valores. Por ejemplo, una expresión sencilla podría ser 2 + 3, mientras que una expresión más compleja podría ser x = (2 + 3) * 4 / y, que utiliza varios operadores para realizar una serie de operaciones y asignar el resultado a la variable x.

Operadores:​

Los operadores se utilizan en JavaScript para realizar operaciones en valores o variables. Existen varios tipos de operadores en JavaScript, que se pueden clasificar en las siguientes categorías:

Operadores aritméticos:​

Estos operadores se utilizan para realizar operaciones matemáticas, como suma, resta, multiplicación, división, módulo (resto), incremento y decremento. Aquí tienes algunos ejemplos:

var x = 10;
var y = 5;
var suma = x + y; // Suma de dos valores
var resta = x - y; // Resta de dos valores
var multiplicacion = x * y; // Multiplicación de dos valores
var division = x / y; // División de dos valores
var modulo = x % y; // Módulo de dos valores
var incremento = x++; // Incremento en una unidad del valor de x
var decremento = y--; // Decremento en una unidad del valor de y

Operadores de comparación:​

Estos operadores se utilizan para comparar dos valores o variables y devuelven un valor booleano (true o false), indicando si la comparación es verdadera o falsa. Aquí tienes algunos ejemplos:

var x = 10;
var y = 5;
var igualdad = x == y; // Compara si x es igual a y
var desigualdad = x != y; // Compara si x es diferente a y
var mayorQue = x > y; // Compara si x es mayor que y
var mayorIgualQue = x >= y; // Compara si x es mayor o igual que y
var menorQue = x < y; // Compara si x es menor que y
var menorIgualQue = x <= y; // Compara si x es menor o igual que y

Operadores lógicos:​

Estos operadores se utilizan para combinar valores booleanos y devolver un valor booleano resultante. Los operadores lógicos en JavaScript son AND (&&), OR (||) y NOT (!). Aquí tienes algunos ejemplos:

var x = 10;
var y = 5;
var z = 2;
var and = (x > y) && (y > z); // AND lógico de dos expresiones booleanas
var or = (x < y) || (y < z); // OR lógico de dos expresiones booleanas
var not = !(x > y); // NOT lógico de una expresión booleana

Operador lógico AND (&&): El operador lógico AND se representa con dos ampersands (&&). Este operador devuelve true si ambos operandos son true, y false en cualquier otro caso. Por ejemplo:

var a = true;
var b = false;
var resultado = a && b; // El resultado será false, ya que b es false

En este caso, a es true y b es false, por lo tanto, la expresión a && b devuelve false porque ambos operandos no son true.

Operador lógico OR (||): El operador lógico OR se representa con dos barras verticales (||). Este operador devuelve true si al menos uno de los operandos es true, y false si ambos operandos son false. Por ejemplo:

var a = true;
var b = false;
var resultado = a || b; // El resultado será true, ya que a es true

En este caso, a es true y b es false, por lo tanto, la expresión a || b devuelve true porque al menos uno de los operandos es true.

Operador lógico NOT (!): El operador lógico NOT se representa con un signo de exclamación (!). Este operador invierte el valor de un operando booleano. Si el operando es true, el resultado será false, y si el operando es false, el resultado será true. Por ejemplo:

var a = true;
var resultado = !a; // El resultado será false, ya que a es true

En este caso, a es true, y al aplicar el operador ! sobre a, se invierte su valor, por lo tanto, el resultado es false.

Operadores de asignación:​

Estos operadores se utilizan para asignar valores a variables. Aquí tienes algunos ejemplos:

var x = 10; // Asignación de un valor a una variable
var y = 5;
y += x; // Asignación de la suma de dos valores a una variable
y -= x; // Asignación de la resta de dos valores a una variable
y *= x; // Asignación de la multiplicación de dos valores a una variable
y /= x; // Asignación de la división de dos valores a una variable
y %= x; // Asignación del módulo de dos valores a una variable

Operadores de cadena:​

Estos operadores se utilizan para concatenar (unir) dos cadenas de texto. Aquí tienes algunos ejemplos:

var x = "Hola";
var y = "Mundo";
var concatenacion = x + y; // Concatenación de dos cadenas de texto
var concatenacionEspacio = x + " " + y; // Concatenación de dos cadenas de texto con un espacio

Operadores de tipo:​

Estos operadores se utilizan para comprobar el tipo de un valor o variable. Aquí tienes algunos ejemplos:

var x = 10;
var y = "Hola";
var esNumero = typeof x === "number"; // Comprueba si x es un número
var esCadena = typeof y === "string"; // Comprueba si y es una cadena de texto

Operadores condicionales ternarios:​

Estos operadores se utilizan para realizar una evaluación condicional. Se trata de una forma abreviada de escribir una sentencia if-else. Aquí tienes algunos ejemplos:

var x = 10;
var y = 5;
var resultado = x > y ? "x es mayor que y" : "x es menor o igual que y"; // Evalúa una condición y asigna un valor a una variable

Expresiones:​

Las expresiones son combinaciones de valores, variables y operadores que producen un valor. En JavaScript, las expresiones pueden ser tan simples o complejas como se necesite. Aquí tienes algunos ejemplos de expresiones en JavaScript:

var nombre = "Juan"; // Una expresión que asigna la cadena "Juan" a la variable nombre
var saludo = "Hola " + nombre + "!"; // Una expresión que combina varias cadenas usando el operador de concatenación (+)
var esMenorDeEdad = edad < 18; // Una expresión que compara la variable edad con el valor 18 y devuelve un valor booleano
var resultado = (3 + 4) * (5 - 2); // Una expresión que realiza varias operaciones aritméticas y asigna el resultado a la variable resultado
var esPar = numero % 2 == 0; // Una expresión que calcula el resto de la división de la variable numero por 2 y comprueba si es igual a cero

Es importante tener en cuenta que las expresiones en JavaScript se evalúan de izquierda a derecha y de acuerdo a las reglas de precedencia de operadores. También es posible utilizar paréntesis para forzar un orden de evaluación específico. Por ejemplo, en la expresión 3 + 4 5, primero se realiza la multiplicación y luego la suma, porque el operador de multiplicación tiene mayor precedencia que el operador de suma. Si se desea cambiar el orden de evaluación para realizar primero la suma, se pueden utilizar paréntesis de la siguiente manera: (3 + 4) 5.
