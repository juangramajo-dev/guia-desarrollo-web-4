JavaScript avanzado

Introducción​

El módulo de JavaScript avanzado es una continuación del módulo de fundamentos de JavaScript, en el que se profundiza en temas más avanzados y se exploran nuevas herramientas y técnicas para construir aplicaciones web dinámicas e interactivas.

Entre los temas que se suelen cubrir en el módulo de JavaScript avanzado, se encuentran:

Arrays y objetos: se profundiza en la utilización de estas estructuras de datos en JavaScript, y se aprende a manipular y recorrer los elementos de un array, y las propiedades y métodos de un objeto.

Funciones: se profundiza en el uso de las funciones en JavaScript, incluyendo funciones anónimas, funciones de flecha, funciones recursivas, y funciones de orden superior. También se aborda el tema de los closures y la clausura de funciones.

Eventos y manipulación del DOM: se aprende a utilizar el modelo de eventos de JavaScript para detectar y responder a las acciones del usuario en la página web, y se exploran técnicas para manipular el árbol del DOM (Document Object Model) de la página.

Utilización de JavaScript y Bootstrap para crear una página web interactiva: se utiliza Bootstrap, un framework de diseño web, en conjunto con JavaScript para crear páginas web interactivas que respondan a las acciones del usuario y proporcionen una experiencia de usuario más dinámica.

En resumen, el módulo de JavaScript avanzado se enfoca en profundizar en los conceptos y técnicas básicas de JavaScript, y enseña nuevas habilidades y herramientas que permiten crear aplicaciones web más avanzadas e interactivas.

Comencémos!​

Arrays​

Un array es una colección ordenada de valores, que se identifican mediante un índice numérico. En JavaScript, los arrays pueden contener elementos de diferentes tipos de datos, como números, cadenas de texto, objetos, etc. Para crear un array en JavaScript, se utiliza la siguiente sintaxis:

let miArray = [valor1, valor2, valor3, ...];

Donde valor1, valor2, valor3, etc. son los elementos que se quieren almacenar en el array. Los elementos se separan con comas y se encierran entre corchetes.

Para acceder a un elemento específico de un array, se utiliza su índice numérico, que comienza en cero. Por ejemplo, si queremos acceder al segundo elemento de un array llamado miArray, se haría de la siguiente manera:

let miArray = ["manzana", "naranja", "plátano"];
let segundoElemento = miArray[1]; // "naranja"

En este caso, el índice del segundo elemento es 1, ya que los índices comienzan en cero.

Además de acceder a elementos individuales, es posible recorrer un array completo utilizando diferentes técnicas, como bucles for, for-of, forEach, entre otros.

Objetos​

Un objeto es una colección de pares de claves y valores, donde cada clave se utiliza para identificar un valor específico. En JavaScript, los objetos se representan mediante llaves {} y se utilizan para almacenar y manipular datos más complejos. Para crear un objeto en JavaScript, se utiliza la siguiente sintaxis:

let miObjeto = {
  clave1: valor1,
  clave2: valor2,
  clave3: valor3,
  ...
};

Donde clave1, clave2, clave3, etc. son las claves que se quieren utilizar para identificar los valores, y valor1, valor2, valor3, etc. son los valores que se quieren almacenar.

Para acceder a un valor específico de un objeto, se utiliza la sintaxis objeto.clave. Por ejemplo, si queremos acceder al valor correspondiente a la clave clave2 de un objeto llamado miObjeto, se haría de la siguiente manera:

let miObjeto = { clave1: "valor1", clave2: "valor2", clave3: "valor3" };
let valorDeClave2 = miObjeto.clave2; // "valor2"

En este caso, miObjeto.clave2 devuelve el valor "valor2", que es el valor correspondiente a la clave clave2.

Además de acceder a valores individuales, es posible recorrer un objeto completo utilizando diferentes técnicas, como bucles for-in o utilizando los métodos Object.keys, Object.values y Object.entries.

Métodos de arrays​

JavaScript proporciona una gran cantidad de métodos predefinidos para trabajar con arrays, que permiten realizar operaciones como agregar o eliminar elementos, ordenarlos, filtrarlos, mapearlos, reducirlos, entre otros. Algunos de los métodos más comunes son:

push, pop, shift, unshift: para agregar o eliminar elementos del inicio o final de un array slice, splice: para extraer o reemplazar elementos de un array sort, reverse: para ordenar los elementos de un array filter, map, reduce: para filtrar, mapear o reducir los elementos de un array

push y pop para agregar y eliminar elementos del final de un array:​

let fruits = ['apple', 'banana', 'orange'];

// Agregar un elemento al final del array
fruits.push('mango');
console.log(fruits); // Output: ['apple', 'banana', 'orange', 'mango']

// Eliminar el último elemento del array
fruits.pop();
console.log(fruits); // Output: ['apple', 'banana', 'orange']

shift y unshift para agregar y eliminar elementos del inicio de un array:​

let fruits = ['apple', 'banana', 'orange'];

// Agregar un elemento al inicio del array
fruits.unshift('mango');
console.log(fruits); // Output: ['mango', 'apple', 'banana', 'orange']

// Eliminar el primer elemento del array
fruits.shift();
console.log(fruits); // Output: ['apple', 'banana', 'orange']

slice y splice para extraer o reemplazar elementos de un array:​

let fruits = ['apple', 'banana', 'orange', 'mango'];

// Extraer una porción del array
let citrus = fruits.slice(1, 3);
console.log(citrus); // Output: ['banana', 'orange']

// Reemplazar elementos en el array
fruits.splice(1, 2, 'grape', 'melon');
console.log(fruits); // Output: ['apple', 'grape', 'melon', 'mango']

sort y reverse para ordenar los elementos de un array:​

let numbers = [4, 2, 7, 1, 9];

// Ordenar los elementos del array de forma ascendente
numbers.sort();
console.log(numbers); // Output: [1, 2, 4, 7, 9]

// Ordenar los elementos del array de forma descendente
numbers.reverse();
console.log(numbers); // Output: [9, 7, 4, 2, 1]

filter, map y reduce para filtrar, mapear o reducir los elementos de un array:​

let numbers = [1, 2, 3, 4, 5];

// Filtrar los números pares del array
let evenNumbers = numbers.filter(function (num) {
  return num % 2 === 0;
});
console.log(evenNumbers); // Output: [2, 4]

// Mapear los números del array multiplicándolos por 2
let doubledNumbers = numbers.map(function (num) {
  return num * 2;
});
console.log(doubledNumbers); // Output: [2, 4, 6, 8, 10]

// Reducir los números del array a su suma total
let sum = numbers.reduce(function (accumulator, num) {
  return accumulator + num;
}, 0);
console.log(sum); // Output: 15

Métodos de objetos​

Al igual que con los arrays, JavaScript proporciona una gran cantidad de métodos predefinidos para trabajar con objetos, que permiten realizar operaciones como agregar o eliminar claves, obtener una lista de las claves o valores, entre otros. Algunos de los métodos más comunes son:

Object.keys, Object.values, Object.entries: para obtener una lista de las claves, valores o pares de un objeto Object.assign: para copiar las propiedades de un objeto en otro Object.freeze, Object.seal: para proteger o limitar la modificación de un objeto

Object.keys, Object.values, Object.entries para obtener una lista de las claves, valores o pares de un objeto:​

var persona = {
  nombre: 'Juan',
  edad: 30,
  ciudad: 'Madrid'
};

// Obtener las claves del objeto
var claves = Object.keys(persona);
console.log(claves); // Salida: ['nombre', 'edad', 'ciudad']

// Obtener los valores del objeto
var valores = Object.values(persona);
console.log(valores); // Salida: ['Juan', 30, 'Madrid']

// Obtener los pares [clave, valor] del objeto
var pares = Object.entries(persona);
console.log(pares); // Salida: [['nombre', 'Juan'], ['edad', 30], ['ciudad', 'Madrid']]

Object.assign para copiar las propiedades de un objeto en otro:​

var obj1 = { a: 1, b: 2 };
var obj2 = { c: 3, d: 4 };

// Copiar las propiedades de obj2 en obj1
var obj3 = Object.assign({}, obj1, obj2);
console.log(obj3); // Salida: { a: 1, b: 2, c: 3, d: 4 }

Object.freeze y Object.seal para proteger o limitar la modificación de un objeto:​

var persona = {
  nombre: 'Juan',
  edad: 30
};

// Congelar el objeto, evitando cualquier modificación
Object.freeze(persona);

persona.edad = 35; // No tiene efecto
console.log(persona); // Salida: { nombre: 'Juan', edad: 30 }

// Sellar el objeto, permitiendo modificar los valores existentes pero no agregar ni eliminar propiedades
Object.seal(persona);

persona.edad = 35; // Modificación permitida
persona.ciudad = 'Madrid'; // No tiene efecto
delete persona.nombre; // No tiene efecto
console.log(persona); // Salida: { nombre: 'Juan', edad: 35 }

Conclución​

Los arrays y los objetos son dos estructuras de datos importantes en JavaScript que se utilizan para almacenar y manipular colecciones de datos. Los arrays se utilizan para colecciones de elementos ordenados, mientras que los objetos se utilizan para colecciones de pares de claves y valores.

En el ámbito de la programación avanzada en JavaScript, es importante comprender en detalle cómo funcionan los arrays y los objetos. Además, existen diversas técnicas y funcionalidades avanzadas que se pueden utilizar con arrays y objetos para crear aplicaciones web interactivas y eficientes.
