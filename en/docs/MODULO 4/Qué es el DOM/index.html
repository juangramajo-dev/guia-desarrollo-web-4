¿Qué es el DOM?

El DOM (Document Object Model) o Modelo de Objetos del Documento es una interfaz de programación que nos permite crear, cambiar o remover elementos del documento HTML. También podemos agregar eventos a esos elementos para hacer nuestra página más dinámica.

El DOM visualiza el documento HTML como un árbol de nodos, donde cada nodo representa un elemento del documento.

Echemos un vistazo a este código HTML para entender mejor la estructura de árbol del DOM:

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Estructura de árbol del DOM</title>
  </head>
  <body>
    <h1>Estructura de árbol del DOM</h1>
    <h2>Aprende sobre el DOM</h2>
  </body>
</html>

En este código, el documento HTML es el nodo raíz, que contiene un nodo hijo, el elemento <html>. A su vez, el elemento <html> contiene dos nodos hijos, que son los elementos <head> y <body>. Tanto el elemento <head> como el elemento <body> pueden tener sus propios nodos hijos.

Aquí hay otra forma de visualizar este árbol de nodos:

Document
└── <html>
    ├── <head>
    └── <body>

Podemos acceder a estos elementos en el documento y hacer cambios en ellos utilizando JavaScript.

Cómo seleccionar elementos en el documento​

Existen diferentes métodos para seleccionar elementos en el documento HTML. En este artículo, nos enfocaremos en tres de esos métodos: getElementById(), querySelector() y querySelectorAll().

getElementById()​

En HTML, los IDs se utilizan como identificadores únicos para los elementos. Esto significa que no podemos tener el mismo ID para dos elementos diferentes.

Por ejemplo, el siguiente código sería incorrecto:

<p id="para">Este es mi primer párrafo.</p>
<p id="para">Este es mi segundo párrafo.</p>

En su lugar, debemos asegurarnos de que los IDs sean únicos, como se muestra a continuación:

<p id="para1">Este es mi primer párrafo.</p>
<p id="para2">Este es mi segundo párrafo.</p>

En JavaScript, podemos seleccionar un elemento HTML haciendo referencia a su ID de la siguiente manera:

const elemento = document.getElementById('miElemento');

Este código le indica al navegador que obtenga el elemento <p> que tiene el ID para1 y lo imprima en la consola.

const paragraph1 = document.getElementById('para1');
console.log(paragraph1);

Si deseamos leer solo el contenido del párrafo, podemos utilizar la propiedad textContent dentro de console.log():

const paragraph1 = document.getElementById('para1');
console.log(paragraph1.textContent);

querySelector()​

El método querySelector() nos permite encontrar elementos utilizando selectores de CSS.

Supongamos que tenemos el siguiente código HTML con información sobre nuestros programas de televisión favoritos:

<h1>Programas de TV Favoritos</h1>
<ul class="lista">
  <li>Golden Girls</li>
  <li>Archer</li>
  <li>Rick and Morty</li>
  <li>The Crown</li>
</ul>

Si deseamos seleccionar y imprimir el elemento <h1> en la consola, podemos utilizar el nombre de la etiqueta dentro de querySelector():

const h1Elemento = document.querySelector('h1');
console.log(h1Elemento);

Si queremos apuntar al elemento con la clase lista para imprimir la lista no ordenada en la consola, podemos utilizar .lista dentro de querySelector():

El punto antes de lista le indica al navegador que apunte al nombre de la clase. Si deseas apuntar a un ID, entonces se utiliza el símbolo # antes del nombre.

const lista = document.querySelector('.lista');
console.log(lista);

querySelectorAll()​

Este método encuentra todos los elementos que coinciden con el selector de CSS y devuelve una lista (NodeList) de esos nodos.

Si queremos encontrar todos los elementos <li> en nuestro ejemplo, podemos utilizar el combinador de hijos > para seleccionar todos los elementos hijos de <ul>:

const listaItems = document.querySelectorAll('ul > li');
console.log(listaItems);

Si deseamos imprimir los elementos reales <li> con los programas de TV, podemos utilizar el ciclo forEach() para iterar sobre la NodeList e imprimir cada uno de los elementos:

const listaDeElementos = document.querySelectorAll('ul > li');

listaDeElementos.forEach((item) => {
  console.log(item);
});

Cómo agregar nuevos elementos al documento​

Podemos utilizar el método document.createElement() para

agregar nuevos elementos al árbol del DOM.

Veamos este ejemplo:

<h1>Razones por las que amo freeCodeCamp:</h1>

En este caso, solo tenemos una etiqueta <h1> en la página, pero queremos agregar una lista de razones por las que amamos freeCodeCamp debajo de esa etiqueta <h1> usando JavaScript.

Primero, creamos un elemento <ul> utilizando document.createElement(). Asignamos eso a una variable llamada listaSinOrden:

let listaSinOrden = document.createElement('ul');

Luego, necesitamos agregar ese elemento <ul> al documento utilizando el método appendChild():

document.body.appendChild(listaSinOrden);

El siguiente paso es agregar un par de elementos <li> dentro del elemento <ul> utilizando el método createElement():

let elemento1Lista = document.createElement('li');

let elemento2Lista = document.createElement('li');

Después, utilizamos la propiedad textContent para agregar texto a nuestros elementos de lista:

let elemento1Lista = document.createElement('li');
elemento1Lista.textContent = "Es gratuito";

let elemento2Lista = document.createElement('li');
elemento2Lista.textContent = "Es increíble";

El último paso es utilizar el método appendChild() para agregar los elementos de lista al elemento <ul>:

let elemento1Lista = document.createElement('li');
elemento1Lista.textContent = "Es gratuito";
listaSinOrden.appendChild(elemento1Lista);

let elemento2Lista = document.createElement('li');
elemento2Lista.textContent = "Es increíble";
listaSinOrden.appendChild(elemento2Lista);

Así es como queda el código completo:

let listaSinOrden = document.createElement('ul');
document.body.appendChild(listaSinOrden);

let elemento1Lista = document.createElement('li');
elemento1Lista.textContent = "Es gratuito";
listaSinOrden.appendChild(elemento1Lista);

let elemento2Lista = document.createElement('li');
elemento2Lista.textContent = "Es increíble";
listaSinOrden.appendChild(elemento2Lista);

Cómo usar la propiedad style para cambiar estilos en línea de CSS​

La propiedad style nos permite cambiar los estilos CSS en línea de los elementos del documento.

En este ejemplo, vamos a cambiar el color del texto de un elemento <h1> de negro a azul utilizando la propiedad style.

Aquí está nuestro HTML:

<h1>Fui cambiado a azul usando JavaScript</h1>

Primero, seleccionamos el elemento <h1> utilizando el método querySelector():

const h1 = document.querySelector('h1');

Luego, utilizamos h1.style.color para cambiar el color del texto a azul:

const h1 = document.querySelector('h1');
h1.style.color = 'blue';

Así se ve el resultado en el navegador:

Podemos utilizar esta propiedad style para cambiar diversos estilos CSS en línea, como background-color, border-style, font-size, entre otros.

Cómo usar addEventListener() para escuchar eventos en la página​

El método addEventListener() nos permite agregar un evento a un elemento HTML, como un botón.

En este ejemplo, cuando se hace clic en un botón, se muestra un mensaje de alerta.

En nuestro HTML, tenemos un elemento <button> con el ID btn:

<button id="btn">Mostrar Alerta</button>

Podemos seleccionar ese elemento en JavaScript utilizando el método getElementById() y asignándolo a una variable llamada boton:

const boton = document.getElementById('btn');

Luego, utilizamos addEventListener() para agregar el evento al botón. El evento será un clic (click), y la función dentro de addEventListener() activará el mensaje de alerta:

boton.addEventListener('click', () => {
  alert('Gracias por hacer clic');
});

Así es como se ve el código completo:

const boton = document.getElementById('btn');

boton.addEventListener('click', () => {
  alert('Gracias por hacer clic');
});

Ahora, cuando se hace clic en el botón, aparecerá el mensaje de alerta.

¡Claro! Aquí tienes la información adicional sobre el target, value e innerHTML en el contexto del DOM en JavaScript:

Propiedad target​

La propiedad target se utiliza en el contexto de los eventos para hacer referencia al elemento objetivo o destino del evento. Por ejemplo, al manejar un evento de clic en un botón, el evento contiene información sobre el elemento que se hizo clic, y puedes acceder a él a través de la propiedad target.

button.addEventListener('click', function(event) {
  console.log(event.target);
});

En este ejemplo, al hacer clic en un botón, se imprime en la consola el elemento que fue el objetivo del evento.

Propiedad value​

La propiedad value se utiliza comúnmente en elementos de formulario, como <input> y <select>, para obtener o establecer el valor ingresado o seleccionado por el usuario. Puedes acceder a esta propiedad para obtener el valor actual del elemento o modificarlo.

const input = document.getElementById('myInput');
console.log(input.value); // Obtener el valor actual del elemento

input.value = 'Nuevo valor'; // Establecer un nuevo valor al elemento

En este ejemplo, la propiedad value se utiliza para obtener el valor actual de un campo de entrada y también para establecer un nuevo valor en él.

Propiedad innerHTML​

La propiedad innerHTML se utiliza para obtener o establecer el contenido HTML de un elemento. Puedes acceder a esta propiedad para obtener el contenido actual del elemento o modificarlo.

const element = document.getElementById('myElement');
console.log(element.innerHTML); // Obtener el contenido HTML actual del elemento

element.innerHTML = '<p>Nuevo contenido</p>'; // Establecer un nuevo contenido HTML en el elemento

En este ejemplo, la propiedad innerHTML se utiliza para obtener el contenido HTML actual de un elemento y también para establecer un nuevo contenido HTML en él.

Cómo usar el DOM en proyectos del mundo real​

Esto fue solo una breve introducción a algunos de los métodos del DOM que podemos utilizar. Hay muchos más ejemplos y técnicas que no se cubrieron en este artículo.

Si deseas comenzar a construir proyectos de JavaScript para principiantes y trabajar con el DOM, te recomiendo revisar el artículo "40 proyectos de JavaScript para principiantes".

Conclusión​

El DOM (Modelo de Objetos del Documento) es una interfaz de programación que nos permite crear, cambiar o remover elementos del documento HTML. También podemos agregar eventos a esos elementos para hacer nuestra página más dinámica.

En JavaScript, podemos seleccionar elementos del DOM utilizando métodos como getElementById(), querySelector() y querySelectorAll(). Podemos agregar nuevos elementos al documento utilizando document.createElement(). También podemos cambiar estilos en línea utilizando la propiedad style. Y, por último, podemos agregar eventos a elementos utilizando addEventListener().

Espero que hayas disfrutado de este artículo y te deseo mucho éxito en tu viaje de aprendizaje de JavaScript.
